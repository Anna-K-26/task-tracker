# Агенты и структура проекта

В этом проекте используются специализированные агенты и навыки для автоматизации разработки.

## 🏗 Структура проекта

Проект представляет собой веб-приложение для управления задачами:

*   **Backend**: `main.py` на базе FastAPI.
*   **Frontend**: Ванильный JavaScript (`static/script.js`), CSS (`static/styles.css`) и HTML-шаблоны (`templates/index.html`).
*   **Данные**: Хранятся в локальных JSON-файлах (`tasks.json`, `users.json`, `stages.json`).
*   **Инструментарий**: `.cursor/` содержит правила (`rules/`), навыки (`skills/`) и настройки агентов (`agents/`).

## 🛠 Инструменты разработки

**ВАЖНО**: Для управления зависимостями и запуска проекта всегда следует использовать **uv**.

*   Установка зависимостей: `uv sync`
*   Запуск проекта: `uv run main.py`

## 🔌 Интеграция MCP (Model Context Protocol)

В проекте настроены и должны активно использоваться следующие MCP-серверы:

*   **Context7**: Для получения актуальной документации по библиотекам (FastAPI, Pydantic и др.). Всегда используйте его вместо того, чтобы полагаться на встроенные знания о внешних пакетах.
*   **Playwright**: Для тестирования интерфейса, создания скриншотов и автоматизации действий в браузере.

## Как использовать

1.  **MCP**: Агенты должны **всегда** обращаться к доступным MCP-инструментам для получения актуальной информации или взаимодействия с внешними системами (браузером, документацией).
2.  **Навыки**: Напишите в чате: "используй скилл [название]".
3.  **Запуск**: Всегда используйте `uv` для команд в терминале.
