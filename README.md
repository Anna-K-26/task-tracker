# Task Tracker (Канбан-доска и Диаграмма Ганта)

Простое и эффективное приложение для управления задачами, построенное на **FastAPI** и ванильном **JavaScript**.

## 🚀 Основные возможности

- **Канбан-доска**: Визуальное управление задачами через колонки "К работе", "В работе" и "Готово".
- **Диаграмма Ганта**: Визуализация временных рамок задач с возможностью изменения масштаба (день, неделя, месяц).
- **Drag-and-Drop**: Перетаскивание задач между колонками на доске.
- **Интерактивный Гант**: Изменение дат начала и окончания задач прямо на диаграмме (resize).
- **Сортировка и фильтрация**: Сортировка задач по приоритету или датам, фильтрация по ответственному.
- **Управление задачами**: Создание, редактирование и удаление задач через удобные модальные окна.

## 🛠 Технологический стек

- **Backend**: FastAPI (Python 3.10+)
- **Frontend**: HTML5, CSS3 (Custom Styles), JavaScript (Vanilla JS)
- **Хранение данных**: Локальный JSON-файл (`tasks.json`)
- **Иконки**: Font Awesome 6.0

## ⚙️ Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone <url-репозитория>
   cd таск-трекер
   ```

2. **Установите зависимости**:
   Рекомендуется использовать виртуальное окружение.
   ```bash
   pip install fastapi jinja2 requests uvicorn
   ```
   *Или используйте `uv`:*
   ```bash
   uv sync
   ```

3. **Запустите сервер**:
   ```bash
   python main.py
   ```
   Приложение будет доступно по адресу: [http://127.0.0.1:8000](http://127.0.0.1:8000)

## 📝 Использование

- **Добавление задачи**: Нажмите кнопку "+" в любой колонке канбан-доски.
- **Переключение вида**: Используйте кнопки в верхней части экрана для переключения между доской и диаграммой Ганта.
- **Редактирование**: Нажмите кнопку "Open" на карточке задачи или кликните по полоске на диаграмме Ганта.
- **Изменение сроков**: На диаграмме Ганта потяните за края полоски задачи, чтобы изменить её длительность.
